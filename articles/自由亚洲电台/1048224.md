[香港钟翰林被捕前 曾试图向台湾求援](https://www.rfa.org/mandarin/yataibaodao/gangtai/hcm2-07302020160121.html)
------

<p> </p><p>香港前学生动源召集人钟翰林与其他3名成员，本月29号日因触犯《香港国安法》被捕。钟翰林21日就已经担心自身的安危，曾经询问过律师是否有离开香港的必要，并打电话向台湾“人道援助关怀行动专案”求助。不过，因为疫情卡关未能成行，最后不幸在香港被捕。<br/> <br/> 钟翰林最后一次现身台湾是在今年1月、武汉肺炎疫情爆发之前。当时他特别呼吁蔡政府，香港人很希望在“港澳条例”18条下，如何申请政治庇护，让香港人在台湾得到真正的帮助。“因为除了《难民法》之外，我们虽然知道可以通过这（第18条）来申请，但是台湾政府也没有给我们完善的条件，可以来台湾申请政治庇护。”</p><p> </p><ul><li><b><a class="external-link" <br/> 黄燕茹说，现在因为疫情关系，香港人只有依亲、商务履约、读书、投资移民等可以入境台湾，但是，香港抗争手足年轻化，年轻人没有条件依照商务履约来台。加上现在暴动罪，一提堂护照很可能就被没收。<br/> <br/> 黄燕茹：“陆委会跟移民署也没有怎么搭配的问题，政治庇护现在也没有任何依据，到底什么样的定义叫立即性危险，到底什么状况可以进来。如果依照商务履约跟依亲进来，这不是不需要这个办公室了吗？”</p><p> </p><p><div class="image-inline captioned" style="width:1280px;"><div style="width:1280px;"><img 钟翰林来台期间，独眼新闻创办人Nancy多次陪同钟翰林拜会台湾的政府单位。Nancy提到钟翰林在21日出庭前，曾经询问是不是有可能来台湾。Nancy说，今年一月钟翰林最后一次来台时，仍坚持要返回香港。但是，局势变化太快，《香港国安法》通过之后到出庭之间，短短不到三周的时间，钟翰林征询过律师是否有离开香港的必要，律师告诉他有这个必要。当时钟翰林已经计划离港，最后希望放在台湾“人道援助关怀行动专案”。<br/> <br/> Nancy：“他的情况很紧急，我告诉他你赶快打到办公室问，因为香港跟台湾都有办公室。香港办公室告诉他，你的情况要打到台湾，这个电话是给台湾人用。台湾办公室告诉他，现在疫情还是无法进来（台湾）。他就知道自己现阶段无法来。”<br/> <br/> 陆委会回复的质询本台说，钟翰林从未向台湾提出庇护的相关申请，人道救援相关专线也并未接获钟翰林的相关查询电话。陆委会发言人邱垂正30日在记者会提到，“台港服务交流办公室”，迄今已有超过一千余件询问案件，主要询问来自来台移民定居最多，其次是投资、就业与就学等事宜。<br/> <br/> <br/> 自由亚洲电台记者黄春梅 台北报道 责编 许书婷 申铧  网编 瑞哲</p>
